spotify-to-csv-export-webapp

Spotify cant save a user's music library. This application implements this missing feature.

How it works?

  1. After you click "Sign in with Spotify" -> redirect to Spotify account service login page.
    * application require access to your Spotify account to see your library.
  2. If the security check is passed and access is granted -> redirect to the page with your Spotify library.
    * application receives your access token, which is stored for ~7 days or can be deleted once you "sign out".
  3. After clicking on "sign out" -> the access token is deleted.

Used scopes:

  • user-read-private: to get Spotify userid and username
  • user-library-read: to read the spotify user's library
  • playlist-read-collaborative: for reading collaborative playlists
  • playlist-read-private: to read private playlists
